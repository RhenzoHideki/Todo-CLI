## Rust To-do CLI 
A rust todo list CLI in rust

## Build 
```bash
# Clone the repository to your local host
git clone

# Change Directory
cd TODO-CLI 

# Compile the program
rustc src/main.rs -o todo

```
## Usage
```bash
todo add <task_name>
todo complete <task_name>
todo remove <task_name>
todo list ""

```
